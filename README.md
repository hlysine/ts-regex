## Installation

Install ts-regex with your preferred package manager, and then import it with CommonJS or ES Modules syntax:

```bash
npm install ts-regex

yarn add ts-regex
```

```js
import { createRegex } from 'ts-regex';

const { createRegex } = require('ts-regex');
```
